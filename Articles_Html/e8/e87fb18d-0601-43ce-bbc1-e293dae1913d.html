Description

Map Windows groups to WCF service methods using the WCF PrincipalPermission attribute.&nbsp; Incoming client username credentials will be mapped to the associated Windows group.&nbsp; Service method access will be granted to a user only if they are a member of the group associated with the service method being called.

The following example demonstrates how the WCF service &#8220;Add&#8221; will only run for users belonging to the &#8220;CalculatorClients&#8221; Windows group.

// Only members of the CalculatorClients group can call this method.
&#91;PrincipalPermission(SecurityAction.Demand, Role = "CalculatorClients")&#93;
public double Add(double a, double b)

&nbsp;&nbsp;&nbsp; return a &#43; b;
