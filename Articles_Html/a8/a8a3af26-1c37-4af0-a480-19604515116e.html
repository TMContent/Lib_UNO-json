Applies to

  • PCI DSS Requirement 2.2.a

What to Check For

The system configuration standards should be available for each type of system on the network and should be consistent with industry-accepted standards. Industry-accepted standards include the hardening guides published by SANS, NIST, CIS and NSA.

Why

It is prudent to use configuration standards that have been subject to peer-review and have been proven effective by information security professionals who have already faced the challenges of standardizing system configurations.

How to Check

Procure copies of the relevant hardening guides from SANS, NIST, CIS, and NSA. Compare the company system configuration standards to the industry-accepted standards. Make sure that the company system configuration standards are similar or identical to the industry-accepted standards.

How to Fix

Procure copies of the relevant hardening guides from SANS, NIST, CIS, and NSA. Use these industry-accepted configuration guides as the foundation for the company's configuration standards. Prepare configuration standards for each type of system used on the network. Use the configuration standards when setting up new systems. When changes are made to the configuration standards, apply the corresponding changes to the existing systems.
