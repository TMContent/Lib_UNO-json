Applies to

  • PCI DSS Requirement 8.1

What to Do

Give each user a unique account.

Why

Giving each user a unique account allows managing access control effectively.

How

Perform the following actions to make a new user account for each authorized user that does not have one in a Microsoft Active Directory environment:

  1. Launch the user management console. Launch the user management application by clicking on "Start", "Programs", "Administrative Tools", "Active Directory Users and Computers".

  2. Create a new user account. To create a new user, navigate to the user list, right click on it, select "New", "User"

  3. Configure the new user account. Fill in the user account information in the "New Object - User" dialog. Configure group membership and data access control permissions for the new user account. Perform other actions associated with creating new user accounts, such as adding e-mail accounts, allocating personal work-space, etc.
