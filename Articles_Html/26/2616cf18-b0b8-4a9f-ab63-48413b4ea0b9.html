Applies to

  • PCI DSS Requirement 7.1.2

What to Do

Grant the minimum data access privileges that are needed for each individual to perform his or her job functions.

Why

The least privilege principle helps prevent unauthorized access.

How

Use the following steps to implement the data access policy using the lease privilege principle:

  1. Identify job functions. Identify the job functions of each employee.

  2. Identify access requirements. Identify the data access required for each employee. Define the data access privileges for each employee based on job responsibilities.

  3. Get management approval. Have the management review and authorize the data-access privilege specifications.

  4. Configure ACLs. Once approved, implement the data access control systems.

  5. Update documentation. Attach the access authorization documents to the network documentation.
