Applies to

  • PCI DSS Requirement 3.4.1.a
  • PCI DSS Requirement 3.4.1.b

What to Do

Use a secure authentication mechanism for disk encryption that is separate from the rest of the network. Store the disk encryption keys securely, for example on removable media that is protected with strong access controls and physical security.

Why

Disk encryption should use its own authentication mechanism to make it more difficult for a potential attacker to get the encryption key, especially in the event of an insider attack.

How

Decide whether to use whole disk encryption. Identify a disk encryption application that uses its own secure authentication mechanism, such as a password or a key file. Install and configure the chosen disk encryption suite. Secure the disk encryption keys with strong access controls and physical security.
