What to Check For

An employee security awareness program should be conducted.

Why

An employee security awareness program increases staff participation in maintaining a secure computing environment.

How to Check

Make sure the employee security awareness program is conducted in compliance with PCI DSS by using the following steps:

  1. Verify program existence. Verify that a security awareness program exists and is available to all employees by examining the documentation of the program.

  2. Examine security awareness training materials. Verify that the security awareness program uses multiple communications media such as posters, letters, web sites, meetings, etc. by examining the program materials.

  3. Examine records of security awareness training. Verify that all employees engage in security awareness training upon hire and at least once a year after that by interviewing a random sample of employees and examining records of conducted training.

  4. Check employee acknowledgement forms. Examine the information security documentation for the presence of a signed form for each employee that states that the employee has attended an information security awareness session and understands the company information security policy.

How to Fix

Establish and conduct a security awareness program by performing the following tasks:

  1. Establish a security awareness training program. Delegate responsibility for the program to a qualified staff member. Develop training materials. Designate staff responsible for training. Schedule yearly training sessions. Train new employees upon hire. Make a spreadsheet with a list of all employees and the date of their most recent security awareness training to make sure all employees participate. Manage a portfolio of security training materials. Have the person responsible for training produce or procure these materials.  These materials should be in multiple formats such as presentations, letters, posters, videos, etc.

  2. Prepare employee acknowledgement forms. Prepare a form that states that the employee who signed understands the information security policy. Print this form and make it available during each training session. After each training session hand it out to the employees to sign. Collect the signed forms and attach them to the information security documentation. Discard the older sign-off forms if you have more than one for an employee.

  3. Document the security training program. Keep records of conducted training. Keep the signed acknowledgment forms. Schedule presentations and/or annual training sessions. Maintain a spreadsheet of the most recent security awareness training for each employee.
