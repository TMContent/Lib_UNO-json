What to Check For

If TLS is used, only trusted certificates or keys should be accepted.

Why

Using a trusted key or certificate prevents an attacker from posing as a trusted party or substituting the key to be able to decrypt the transmission.

How to Check

Identify whether TLS is used. Identify all the certificates and keys used to encrypt the transmissions. Make sure that all the used keys and certificates are trusted by examining the certificates and keys and interviewing the issuing parties.

How to Fix

Identify whether TLS is used. Identify all the certificates and keys used to encrypt the transmissions. Verify that all the used keys and certificates are authorized by examining the certificates and keys and interviewing the issuing parties. Add all the used keys and certificates to the trusted certificates or keys list of each respective application, such as a web browser.
