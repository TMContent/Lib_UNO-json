Applies to

  • PCI DSS Requirement 10.2.6

What to Check For

Initialization of logs should be logged.

How to Check

Examine the logs on the log server to verify that they have log entries that mark the initialization of logging. If log initialization has not happened for a long time, it may be more convenient to restart the log server to verify that it creates a log entry upon initialization. If it is not acceptable to restart the log server, it may be sufficient to consult the vendor's documentation or contact technical support for confirmation that the log server solution logs initialization.

How to Fix

This functionality is implemented and enabled by default on most log generating products, including Microsoft Windows Operating Systems. It is unlikely that there is a configuration setting for this, but if your device does not log initialization, please refer to the vendor's documentation or technical support for instructions on enabling it.
