Applies to

  • PCI DSS Requirement 3.5
  • PCI DSS Requirement 3.5.2.a

What to Check For

Verify that keys stored in configuration files are encrypted. Ensure that keys used to encrypt other keys are stored separately from the keys used to encrypt data.

Why

The keys should be encrypted when stored to prevent an attacker gaining access to them in the event of a successful compromise.

How to Check

Find where the keys are stored. Make sure that the keys are encrypted when they are stored. Make sure that the keys used to encrypt other keys are stored separately from the keys used to encrypt data.

How to Fix

Find where the encryption keys used to encrypt data are stored. Choose an encryption suite that provides secure cryptography. Use the chosen encryption application to secure the stored keys. Designate a separate secure storage location for the key-encrypting keys.
