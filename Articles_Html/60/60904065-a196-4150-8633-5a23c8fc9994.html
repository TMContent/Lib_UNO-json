What to Check For

Policy should define information security responsibilities for employees and contractors.

Why

Information security responsibilities should be explicitly defined so that people can understand what is expected of them.

How to Check

Examine the information security documentation to make sure it includes explicit definitions for the following requirements:

  • Writing and distributing information security policies and procedures
  • Monitoring logs and security alerts, and distributing information to the appropriate staff
  • Managing incident response procedures
  • Managing users, including adding, changing, and removing user accounts
  • Managing data access control

How to Fix

Define the following responsibilities and include the definitions in the information security policy:

  • Writing and distributing information security policies and procedures
  • Monitoring logs and security alerts, and distributing information to the appropriate staff
  • Managing incident response procedures
  • Managing users, including adding, changing, and removing user accounts
  • Managing data access control
