Applies to

PCI DSS Requirement 3.6.5.a

What to Do

Retire old encryption keys. Do not use encryption keys that have been used before.

Why

Retiring old keys ensures that unauthorized disclosure of old keys by former employees does not facilitate a compromise.

How

Use the following steps to make sure that old encryption keys are retired:

  1. Define procedures for managing old encryption keys. Define a procedure for archiving old encryption keys. Define a procedure for revoking old encryption keys.

  2. Implement procedures for managing old encryption keys. Implement the chosen procedures for the archiving and revocation of old encryption keys using technology. You may need to add this functionality to your software if you don't have it already.

  3. Update documentation. Document the mechanisms for archiving and revocation of old encryption keys.
