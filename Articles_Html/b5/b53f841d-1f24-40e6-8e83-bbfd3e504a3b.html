Applies to

  • PCI DSS Requirement 1.3.3

What to Do

Configure the network in a way that does not allow the cardholder data environment to have direct Internet access.

Why

Denying the cardholder data environment Internet connectivity prevents the potential attackers from being able to conduct attacks against it.

How

Remove any routes from the router configuration that allow Internet access to the cardholder data environment. Remove any port forwarding rules from the firewalls and/or routers that allow Internet access to the cardholder data environment. Repeat the How to Check processes to ensure that changes have not accidentally introduced Internet connectivity in the cardholder environment.
