Applies to

  • PCI DSS Requirement 6.4

What to Check For

The change control processes should require a back-out procedure for each change.

Why

The change control processes should require a back-out procedure for each change.

How to Check

Check the change control process documentation to make sure that it requires back-out procedures for each change.

How to Fix

Design and document back-out procedures for all changes before applying them. The back-out procedures should be specific to each application, but they usually involve either uninstalling the update or installing an older, stable version of the application.
