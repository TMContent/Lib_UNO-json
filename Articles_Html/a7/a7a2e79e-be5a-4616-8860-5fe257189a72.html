Applies to

  • Microsoft® Windows Communication Foundation (WCF) 3.5
  • Microsoft Visual Studio® 2008

What to Do

Use platform features where possible to avoid managing keys yourself.

How

For example, by using DPAPI, the encryption key is derived from an account’s password, so Windows handles this for you.
