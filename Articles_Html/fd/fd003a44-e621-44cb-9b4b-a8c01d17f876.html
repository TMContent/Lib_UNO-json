Applies to

  • PCI DSS Requirement 5.2
  • PCI DSS Requirement 5.2.c

What to Check For

The anti-virus software should periodically perform scans to detect known threats.

How to Check

Find the master installation of the anti-virus software and make sure that it has the capability to automatically run scans when deployed. Take a sample of systems that have the anti-virus software installed and check the options to verify that automatic scans are scheduled. To check the anti-virus options, locate the anti-virus administration interface. Some anti-virus packages have control panels on the systems where they are deployed, and some anti-virus suites offer a centralized control panel. The number of options offered by the anti-virus interface is usually small enough that a person with an IT background can figure out what all of them are and how to use them quickly. The administration interface typically has an option to enable automatic scans and set the exact type of the periodic scan. A full scan is usually too time-consuming, so a "quick-scan" that checks the most important system locations is usually sufficient. An occasional full-scan may still be a good idea. Check the logs to make sure that the automatic scans are taking place.

How to Fix

Verify that it is possible to run automatic scans with your anti-virus by referring to the vendor documentation or technical support. The documentation or technical support should provide the exact instructions on how to implement automatic scans. Automatic scans are a basic feature of modern anti-virus packages. If your anti-virus does not offer this feature, please consider installing and configuring a different anti-virus.
