Applies to

  • Microsoft® Windows Communication Foundation (WCF) 3.5
  • Microsoft Visual Studio® 2008

Description

Set the negotiateCredentials attribute to false to use out-of-band credentials. This will require you to provide certificates to the client so they can encrypt and sign messages.

Use the following steps to configure the negotiateCredentials attribute.

  1. Open your app.config or web.config file and set the security mode to Message. 
      <wsHttpBinding>
        <binding name="MessageAndUserName">
          <security mode="Message">
            <message clientCredentialType="UserName" negotiateCredentials=”false” algorithmSuite="Default" />
          </security>
        </binding>
      </wsHttpBinding>
  2. Save the configuration file.
