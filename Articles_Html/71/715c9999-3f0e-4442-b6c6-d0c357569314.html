Applies to

  • PCI DSS Requirement 12.4

What to Do

Define information security responsibilities.

How

Perform the following actions to define information security responsibilities:

  1. Define responsibilities for managing policies and procedures. These responsibilities should include:

    • Creating information security policies and procedures
    • Distributing information security policies and procedures
    • Updating information security policies and procedures

  2. Define responsibilities for monitoring logs and alerts. These responsibilities should include:

    • Monitoring logs and alerts
    • Analyzing security information
    • Distributing security information, such as warnings and status updates, to information security and management staff

  3. Define responsibilities for incident response procedures. These responsibilities should include:

    • Creating incident response procedures
    • Distributing incident response procedures

  4. Define responsibilities for user management. These responsibilities should include:

    • Creating user accounts
    • Modifying user accounts
    • Removing user accounts
    • Implementing authentication technologies

  5. Define responsibilities for data access control. These responsibilities should include:

    • Management of access authorization forms
    • Implementation of access control systems.
    • Configuration of access control systems.
