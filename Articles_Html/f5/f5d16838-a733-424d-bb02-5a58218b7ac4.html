Applies To

  • Web Applications

What to Do

Disable directory browsing for your application's web directories.

Why

Directory browsing allows an attacker to browse your application's directory structure in search of unprotected files and components.

When

Disable directory browsing at all times.

How

Use the following steps to successfully disable directory browsing:

  1. Identify if directory browsing is enabled. Navigate to your application's directories with a web browser. Identify whether your application's server correctly returns an HTTP error message.
  2. Configure the web server to disable directory browsing. The exact procedures for disabling directory browsing are different for different web servers. Please refer to your web server's documentation for detailed instructions for disabling directory browsing.
