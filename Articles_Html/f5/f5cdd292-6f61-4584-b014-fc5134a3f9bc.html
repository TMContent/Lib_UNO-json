Applies to

  • Java

What to Do

Use RSA-2048 whenever strong asymmetric encryption is needed.

Why

Using strong encryption algorithms helps prevent unauthorized disclosure of sensitive information.

How

To use strong asymmetric encryption:

  1. Use Java cryptographic APIs. Use the built-in Java RSA cipher with 2048 bit keys. For an example of using RSA in Java, please see the link at the bottom of the page.

  2. Write cryptographic utility functions that use RSA-2048. Write functions that generate keys, encrypt, and decrypt data.

  3. Use the utility functions. Use the cryptographic utility functions whenever data needs to be encrypted or decrypted using symmetric cryptography.
