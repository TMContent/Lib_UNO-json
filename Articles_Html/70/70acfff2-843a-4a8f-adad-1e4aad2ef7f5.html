Applies to

  • PCI DSS Requirement 1.1.6
  • PCI DSS Requirement 1.1.6.a
  • PCI DSS Requirement 1.1.6.b

What to Check For

The Information Security policy should require the router configuration to be reviewed at least every six months. The results of each review should be included in the router documentation.

Why

Over time, the network configuration and business requirements might change. A regular review is required to make sure that the router configuration accurately matches the business needs while providing an adequate level of security.

How to Check

Check the information security policy to make sure it includes a requirement for a review of the router configuration at least every six months. Check the network documentation to make sure it includes the results of the previous reviews.

How to Fix

Add a point to the information security policy to perform a review of the router configuration every six months. Designate staff responsible for performing the review of the router configuration. Add the results of each review to the network documentation.
