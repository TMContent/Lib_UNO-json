Applies to

  • Microsoft® Windows Communication Foundation (WCF) 3.5
  • Microsoft Visual Studio® 2008

Description

Use the <Security mode> attribute to configure transport security on your binding. See the below example of a wsHttpBinding configured to use transport security:

<bindings>
  <wsHttpBinding>
    <binding name="wsHttpEndpointBinding">
      <security mode="Transport">
      </security>
    </binding>
  </wsHttpBinding>
</bindings>

Transport security is available on all of the bindings except for wsDualHttpBinding.
