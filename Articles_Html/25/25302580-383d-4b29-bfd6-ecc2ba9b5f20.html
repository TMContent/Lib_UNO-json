Applies to

  • PCI DSS Requirement 1.2.3

What to Check For

Firewalls should be installed between any wireless networks and the cardholder data environment. These firewalls should only allow traffic necessary for business purposes.

Why

Wireless access points may be compromised and an attacker may be using them. To prevent sensitive data from leaking out through a compromised wireless network, a firewall should be used to filter traffic between the cardholder data environment and the wireless network.

How to Check

For each wireless network, find the firewall that is supposed to filter traffic between the cardholder data environment and the wireless network. Ensure that the firewall only allows traffic for business purposes. Refer to the vendor’s documentation or technical support with configuration questions.

How to Fix

Install and configure a firewall between each wireless network and the cardholder data environment. Refer to the vendor’s documentation or technical support for installation and configuration instructions.
