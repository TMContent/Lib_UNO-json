What to Do

Use existing libraries to generate keys, encrypt, and decrypt data.

Why

Using strong encryption algorithms helps prevent unauthorized disclosure of sensitive information.

How

The typical scenario that requires web applications to use encryption is protecting stored sensitive data. Symmetric algorithms are the appropriate choice for this requirement. The industry standard for symmetric cryptography is AES with 256-bit key length. To use AES-256 in PHP:

  1. Use the PHP mcrypt extension. Compile PHP with mcrypt support or install PHP with mcrypt support built-in.

  2. Write cryptographic utility functions that use AES-256. Write functions that generate keys, encrypt, and decrypt data. Please, see the link at the bottom of this article for a code example and detailed information about using AES-256 in PHP.

  3. Use the utility functions. Use the cryptographic utility functions whenever data needs to be encrypted or decrypted using symmetric cryptography.
