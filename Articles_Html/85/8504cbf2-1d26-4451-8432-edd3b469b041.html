Applies to

  • PCI DSS Requirement 10.2.1

What to Check For

All user access to cardholder data should be logged.

How to Check

This functionality should be implemented by the centralized log server solution. Examine the logs to verify that access to cardholder data is logged.

How to Fix

This functionality should be implemented by the centralized log server solution. Configure the logging of access to cardholder data using the centralized server solution by following the vendor's instructions.
