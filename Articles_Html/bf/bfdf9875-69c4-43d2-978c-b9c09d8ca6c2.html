Applies to

  • PHP

What to Do

Set the "HTTPOnly" flag on session cookies.

Why

Set the "secure" flag on all session cookies to protect the contents of the session cookies from eavesdropping attacks. Setting the "secure" flag on session cookies ensures they are only transferred over connections that are secured by TLS.

How

To set the "HTTPOnly" flag on session cookies, edit the php.ini file and configure the "session.cookie_httponly" setting, like this:

session.cookie_httponly = On
