What to Check For

Identification and authentication events should be logged.

How to Check

This functionality should be implemented by the centralized log server solution. Examine the logs to verify that identification and authentication events are logged.

How to Fix

This functionality should be implemented by the centralized log server solution. Follow the vendor's instructions to configure the centralized log server to log identification and authentication events.
