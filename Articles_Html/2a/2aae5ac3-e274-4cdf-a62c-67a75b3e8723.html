Applies to

  • PCI DSS Requirement 9.7
  • PCI DSS Requirement 9.7.1
  • PCI DSS Requirement 9.7.2

What to Do

Define a media distribution policy in accordance with PCI DSS. Classify sensitive media as "confidential". Log media distribution. Send sensitive media via secured courier or any other delivery method that can be tracked.

Why

A media distribution policy should be defined to facilitate the making of decisions in a timely and secure manner when media has to be distributed. The policy should cover all types of media storage to ensure that the policy provides adequate guidance for all likely situations. Sensitive media should be classified to make it clear to the recipient that it should not be disclosed. Media distribution should be logged to help audit media distribution events if needed. Media should be sent via secured courier or any other delivery method that can be tracked to assure that the media reaches its recipient, to assist with locating it if it is somehow lost, and to have greater control of the delivery process by being able to monitor it and make calls.

How

Use the following steps to update policy with secure media distribution requirements:

  1. Identify media distribution requirements. Identify the possible scenarios which would require media containing sensitive information to be distributed, for example, an external audit.

  2. Document media distribution procedures. Write down the correct procedures for handling each scenario and add them to the media policy.

  3. Update the policy. Update the information security policy with the media distribution requirements. Add a point to the media policy that requires sensitive media to be classified as "confidential." Add a point to the media policy that requires keeping a log of distributed media. Add a point to the media policy that requires sending sensitive media via secured courier or some other shipping method that can be tracked. Attach the media policy to the information security policy.

  4. Keep a log. Start a log of distributed media that includes the following information: the name of the person authorizing distribution, the name of the intended recipient, the contents of the media, the date, and any other relevant information. Attach the media distribution log to the network documentation.

  5. Comply with the policy. Follow the media distribution policy when distributing sensitive media.
