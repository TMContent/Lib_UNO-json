Applies to

  • PCI DSS Requirement 4.2
  • PCI DSS Requirement 4.2.a
  • PCI DSS Requirement 4.2.b

What to Do

Add a point to the information security policy that forbids sending unencrypted cardholder data. Make an explicit reference to instant messaging software and PANs.

Why

Not using messaging programs to send unencrypted PANs prevents the possibility of them being read by an attacker if intercepted.

How

Add a point to the information security policy that forbids employees from sending unencrypted PANs using instant messaging software. Instruct staff members to not use instant messaging software to send unencrypted PANs.
