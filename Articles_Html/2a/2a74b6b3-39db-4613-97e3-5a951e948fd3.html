Applies to

  • C++

What to Check For

Verify that pointers that are passed as arguments to functions are checked for null before dereferencing.

Why

Dereferencing a null pointer causes the program to crash. Validating pointers before using them prevents unexpected crashes.

How to Check

Search for all functions that accept pointers as arguments. Verify that pointers are checked for null in each function before they are dereferenced.

How to Fix

Identify all functions that accept pointers as arguments. Add code that checks pointers for null before the rest of the program uses these pointers for data access.
