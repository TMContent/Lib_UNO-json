Applies to

  • PCI DSS Requirement 3.4.1.a
  • PCI DSS Requirement 3.4.1.b

What to Check For

Disk encryption should use its own authentication mechanism. The disk encryption keys should be stored securely, such as on a removable media that is protected with strong access controls and physical security.

Why

Disk encryption should use an authentication mechanism separate from the rest of the network to make it more difficult for a potential attacker to get the encryption key, especially in the event of an insider attack.

How to Check

Examine the disk encryption authentication process to make sure it uses credentials different than those used on the network. Examine the disk encryption key storage location to make sure it is protected with strong access controls.

How to Fix

Decide whether to use whole disk encryption. Identify a disk encryption application that uses its own secure authentication mechanism, such as a password or a key file. Install and configure the chosen disk encryption suite. Secure the disk encryption keys with strong access controls and physical security.
