What to Do

Encode XML data. The platform will often take care of this for you, but make sure that it gets done.

Why

Encode data that is included in XML documents for the appropriate context to prevent XML injection vulnerabilities.

How

To encode XML data:

  1. Identify XML data used by the application. Examine the application design to identify all XML data and XML data types used by the application.
  2. Identify code that handles XML data. Review application to find code that uses XML data.
  3. Add data validation methods. Add methods that validate all the data types stored in XML data and used by the application.
  4. Use validation methods. Call the data validation methods from the previous step whenever loading XML data. Validation should be performed before encoding.
  5. Encode XML data. Use the platform's XML encoding functions to encode data used in XML documents. Some XML parsers will encode data fed to them automatically.
