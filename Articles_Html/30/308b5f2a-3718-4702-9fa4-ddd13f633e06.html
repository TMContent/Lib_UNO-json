Applies to

  • PCI DSS Requirement 6.1

What to Check For

The latest vendor-supplied security patches should be installed.

Why

Installing vendor-supplied security patches fixes known security vulnerabilities.

How to Check

Identify the patches installed on each system. Get the most current list of vendor-supplied patches. Make sure that all the applicable security patches supplied by the vendor are installed.

How to Fix

Install the patches in a test environment and test the patches in compliance with PCI DSS Requirement 6.3.&nbsp;Apply all the relevant missing patches to the production environment.&nbsp;Document the results of testing and the update process.
