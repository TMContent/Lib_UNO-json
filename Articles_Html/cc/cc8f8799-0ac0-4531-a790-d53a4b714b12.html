Applies to

  • PCI DSS Requirement 6.4

What to Check For

The change control documentation should include the documentation of customer impact for each change.

Why

Considering the customer impact of each change helps ensure that the customer is satisfied with the results.

How to Check

Examine the change control documentation to make sure that the description of each change includes customer impact considerations.

How to Fix

Revise the change control documentation and add customer impact considerations to the description of each change that doesn't already have it. Consider and document the customer impact of each change in the future.
