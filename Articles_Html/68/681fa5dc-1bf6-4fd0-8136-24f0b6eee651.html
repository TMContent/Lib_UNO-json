What to Do

Bind sockets to specific IP addresses.

Why

Binding sockets to specific IP addresses, instead of INADDR_ANY or 0.0.0.0, prevents socket hijacking.

When

Use this guideline when writing server applications using IP sockets.

How

Bind sockets to specific IP addresses, instead of to INADDR_ANY or 0.0.0.0. For Windows, set the SO_EXCLUSIVEADDRUSE option, which protects against binding to the same port.
