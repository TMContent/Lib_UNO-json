Applies to

  • Java

What to Check For

Verify that only strong TLS algorithms are used.

Why

Using strong encryption algorithms with TLS helps reduce the risk of successful eavesdropping attacks and assures regulatory compliance.

How to Check

To verify that only strong TLS algorithms are used:

  1. Identify the server configuration interface. TLS is usually configured on the web application server. Please, refer to your web application server's documentation or technical support for more information about configuring TLS on your web server.

  2. Examine the server configuration settings. Verify that the web application server is configured to use only FIPS 140-2 compliant algorithms. This means no SSLv1 or SSLv3. In practice, this means using TLSv1 or higher. For more information about compliant algorithms, please see the links at the bottom of this page.

How to Fix

To use strong TLS algorithms, configure the web application server to use TLS and not SSL 3.0 or any earlier version of SSL. The exact instructions for doing so are different for different platforms. Please, refer to your web application server's documentation or technical support for detailed instructions on how to configure the server to use TLS.
