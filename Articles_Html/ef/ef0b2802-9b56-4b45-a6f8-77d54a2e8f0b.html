Applies To

  • ASP.NET 4.0

What to Do

Use forms authentication.

Why

Centralizing authentication mechanisms makes them simpler to maintain.

When

Centralize authentication mechanisms in all web applications that use authentication.

How

ASP.NET has great built-in centralized authentication functionality. To use forms authentication, enable and configure the following section in Web.config:

<system.web>
  <authentication mode="Forms">
     ...
  </authentication>
</system.web>

Additional Resources
