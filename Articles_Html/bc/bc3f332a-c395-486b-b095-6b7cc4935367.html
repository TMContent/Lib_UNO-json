Applies to

  • PHP

What to Check For

Verify that PHP session management is used.

Why

The framework works on a lower level than the application and is therefore better suited to provide session management. PHP offers mature and simple to use session management functionality.

How to Check

To verify that PHP session management is used:

  1. Identify session management functionality. Find code that handles session management.

  2. Verify that PHP session management is used. Review code that uses session management to make sure it uses PHP functions. The easiest way to spot PHP sessions is to look for the session_start() function call.

How to Fix

Refer to the link to the PHP manual below for thorough documentation of PHP session management functionality.
