Applies to

  • PCI DSS Requirement 2.2.1.a

What to Check For

Each server on the network should only have one primary function.

Why

The less functionality a server has, the less surface there is for an attacker to exploit. Furthermore, in the event of physical hardware failure, only one network service per server goes down if each server only implements one primary function.

How to Check

Inspect running network services on each server and make sure only one primary function per server is enabled. Check the network documentation to verify that the stated purpose of the server matches the actual implementation.

How to Fix

Identify all network servers. Identify the primary function of each server. Describe the primary function of each server in the network documentation. Identify the network services and protocols necessary to implement the stated functions. Install and configure the necessary network services and protocols. Disable all other network services and protocols on each server.
