What to Check For

Ensure that your application sets the httpOnlyCookies attribute to true on the authentication cookie to prevent client-side script from accessing the cookie.

How to Fix

Set the httpOnlyCookies attribute on the authentication cookie. Internet Explorer 6 Service Pack 1 supports this attribute, which prevents client-side script from accessing the cookie from the document.cookie property. The System.Net.Cookie class supports an HttpOnly property. The HttpOnly property is always set to true by forms authentication
