What to Check For

Information security responsibilities should be explicitly assigned to specific staff members.

How to Check

Review the information security documentation to verify that it explicitly assigns the following security responsibilities to specific employees and contractors:

  • Responsibility for information security should be assigned to a Chief Security Officer or another qualified member of management
  • Managing security policies and procedures
  • Monitoring security alerts, reviewing logs, and notifying the appropriate staff
  • Managing incident response procedures
  • Managing user accounts
  • Managing data access control

How to Fix

Assign the following responsibilities to formal positions in the company and to contractors:

  • Responsibility for information security should be assigned to a Chief Security Officer or another qualified member of management
  • Managing security policies and procedures
  • Monitoring security alerts, reviewing logs, and notifying the appropriate staff
  • Managing incident response procedures
  • Managing user accounts
  • Managing data access control

It is recommended to assign information security responsibilities to the internal staff, but contractors may be used instead.
