Applies to

  • PHP

What to Do

Configure PHP to use strong session IDs.

Why

Making session IDs cryptographically stronger makes them more difficult to predict and to hijack user sessions.

How

To configure PHP to use strong session IDs, configure the following settings in php.ini:

session.entropy_file = "/dev/urandom"
session.entropy_length = 512

Configuring the settings above tells PHP to use more entropy for generating session IDs, thus making session IDs cryptographically stronger.
