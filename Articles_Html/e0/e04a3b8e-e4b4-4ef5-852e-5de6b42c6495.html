Applies to

  • PCI DSS Requirement 1.1.3

What to Do

Protect each Internet connection with a firewall.

Why

Allowing unfiltered Internet access to the network gives attackers more opportunities to exploit potential vulnerabilities and to control potentially compromised systems remotely.

How

If an Internet connection is not secured by a firewall, install and configure a firewall on that Internet connection. Update the security policy to include the requirement that there be a firewall at each internet connection. Update the network diagram to include any new firewalls.
