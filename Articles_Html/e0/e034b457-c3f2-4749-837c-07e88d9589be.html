What to Check For

Verify that exceptions thrown by input-output functions are handled and error codes are checked.

Why

If errors during file input-output are not handled, unexpected behavior may result leading to a vulnerability.

How to Check

Identify all functions that perform file input-output. Verify that whenever file input-output is performed, code exists that handles possible errors.

How to Fix

Identify all functions that perform file input-output. Identify the possible errors that may result during file IO. Write code that handles exceptions or checks for error codes and responds appropriately.
