Applies to

  • PCI DSS Requirement 3.6.7

What to Do

Manage encryption keys to prevent unauthorized key substitution.

Why

Unauthorized key substitution means that an attacker can substitute one of the authorized keys and gain access to sensitive information without raising alarm.

How

Make a table of authorized keys (known as a keystore) and use only those keys for encryption. Whenever changes are made to the table of keys, make sure the changes are authorized. Document the implementation of the key table and the update mechanism in the key management procedures in the information security policy.
