Applies to

  • Microsoft® Windows Communication Foundation (WCF) 3.5
  • Microsoft Visual Studio® 2008

What to Do

If you need to support interoperability with non-WCF clients, consider using Transport Security.

When

Use this guideline if your non-WCF clients don’t support WS-Security specification.

How

If you have non-WCF clients and if they don’t support WS-Security specification, use Transport security. Message security requires client to understand and support WS-Security specifications and hence won’t work with Non-WCF clients.
