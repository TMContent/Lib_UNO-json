Applies to

  • PCI DSS Requirement 9.9
  • PCI DSS Requirement 9.9.1

What to Check For

Ensure that policy requires media inventories to be performed at least once a year. Verify that inventories are maintained.

Why

Media inventory logs should be kept to assist with management, scheduled media destruction, media distribution, and audits if necessary.

How to Check

Review the media policy to verify that it requires media inventory to be performed at least once a year. Examine the media inventory to make sure it is current and reflects reality.

How to Fix

Add a point to the media policy that requires media inventory to be performed at least once a year. Identify all media storage devices in the cardholder data environment. Number each storage media device. For each storage device in the cardholder data environment, record its number, its location, and its description in the media inventory. Attach the media inventory to the network documentation.
