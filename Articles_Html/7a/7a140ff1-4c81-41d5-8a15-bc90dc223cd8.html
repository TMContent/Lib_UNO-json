Applies to

  • PCI DSS Requirement 1.1.4

What to Check For

The network documentation should include descriptions of the responsibilities for network management roles.

Why

The descriptions of the responsibilities of network management roles help designate the staff responsible for maintaining the security of the network.

How to Check

Review the network documentation to make sure that it includes descriptions of the responsibilities for network management roles.

How to Fix

Designate network management roles, such as network administrators, systems administrators, and information security officers. Assign responsibilities to each network management role. Document the descriptions of network roles and the corresponding responsibilities. Network management responsibilities include:

  • Configuring the firewalls
  • Configuring the routers
  • Configuring the IDS/IPS
  • Reviewing logs
  • Configuring the VPNs
  • Adding, removing, and configuring network devices
