Applies To

  • PCI DSS Requirement 1.1.1

Policy Statement

Include a process for testing and approving changes to the network connections and firewall and router configurations in the firewall and router configuration standards.

Policy Purpose

The firewall and router configuration standards should include a process for testing and approving changes because it helps make sure that the changes have been made correctly and that the new configuration is functional and secure.
