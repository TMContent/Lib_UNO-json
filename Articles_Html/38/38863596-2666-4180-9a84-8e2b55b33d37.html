Applies to

  • PCI DSS Requirement 10.2.3

What to Check For

Access to logs should be logged.

How to Check

This functionality should be implemented by the centralized log server solution. Examine the logs to verify that access to logs is logged.

How to Fix

This functionality should be implemented by the centralized log server solution. Configure the centralized log server to log access to logs. Please refer to the vendor documentation or contact tech support for detailed instructions on configuring your centralized log server for PCI DSS compliance.
