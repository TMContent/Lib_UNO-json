Applies to

  • PCI DSS Requirement 10.2.7

What to Check For

Creation and deletion of logical system objects, such as user accounts or workstations, should be logged.

How to Check

This functionality should be implemented by the centralized log server solution. Examine the logs to verify that creation and deletion of system objects events are logged.  

How to Fix

This functionality should be implemented by the centralized log server solution. Configure the log server to log creation and deletion of system objects by following the vendor's instructions.
