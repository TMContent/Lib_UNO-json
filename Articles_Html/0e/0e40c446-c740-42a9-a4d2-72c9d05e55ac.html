Applies to

  • Java

What to Check For

Verify that sessions are invalidated when users log out.

Why

Invalidating sessions when users log out makes it harder to hijack them.

How to Check

To verify that sessions are invalidated when users log out:

  1. Identify logout functionality. Find code that is responsible for the logout functionality.

  2. Verify that sessions are invalidated upon logout. Look for the following code in functions that are responsible for logging out:

    session.Invalidate();

    where "session" is an HttpSession object.

How to Fix

To invalidate sessions when users log out:

  1. Identify logout functionality. Find code that is responsible for the logout functionality.

  2. Add code to invalidate sessions. Add the following code to the logout function to invalidate the session:

    session.Invalidate();

    where "session" is an HttpSession object.
