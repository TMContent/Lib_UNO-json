Applies To

  • ASP.NET 4.0

What to Check For

Verify that authentication is centralized.

Why

Centralizing authentication mechanisms makes them simpler to maintain.

How To Check

To verify that authentication mechanisms are centralized, make sure that forms authentication is being used. Forms authentication provides a set of centralized authentication functions implemented by the platform. To determine whether forms authentication is being used, search Web.config files for the following string:

<authentication mode="Forms">

It is a good idea to review application code to make sure it takes full advantage of functionality provided by forms authentication.

How To Fix

ASP.NET has great built-in centralized authentication functionality. To use forms authentication, enable and configure the following section in Web.config:

<system.web>
  <authentication mode="Forms">
     ...
  </authentication>
</system.web>
