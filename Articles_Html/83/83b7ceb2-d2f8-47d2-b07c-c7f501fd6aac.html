Applies to

  • PCI DSS Requirement 1.1

What to Check For

The documentation for the firewall configuration should be up-to-date, readily available and cover all the relevant details.

Why

Documenting the firewall configuration is important for maintenance and security purposes. Good firewall documentation makes the PCI DSS compliance certification process simpler and may be required to pass. Documentation is also very useful when making changes to the configuration and when hiring new staff.

How to Check

Make sure that the documentation for the firewall configuration is up-to-date, readily available and covers all the relevant details.

How to Fix

Inspect each firewall and document each configuration setting thoroughly. Delegate responsibility for maintaining up-to-date network documentation to a staff member.
