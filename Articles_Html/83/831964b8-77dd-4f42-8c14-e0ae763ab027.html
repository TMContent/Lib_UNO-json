Applies to

  • Microsoft® Windows Communication Foundation (WCF) 3.5
  • Microsoft Visual Studio® 2008

Description

You can use the SvcTraceViewer.exe utility to view both message log files and trace files. You can find this tool at <Drive Name>:\Program Files\Microsoft SDKs\Windows\v6.0\Bin.

This tool gives you a comprehensive implementation of the step-by-step process of the WCF service, showing each interaction with the WCF runtime and the clients. It shows the object activities, messages, and all errors that occurred in the host’s life. It also provides you a graphical view of the log or trace data.
