What to Do

Do not cache sensitive data.

Why

Do not cache sensitive data to prevent attackers from being able to recover it in the event of a compromise.

How

To prevent caching of sensitive data:

  1. Identify caching code. Find code that caches data.

  2. Identify sensitive data. Identify the types of sensitive data handled by your application.

  3. Review caching code. Review the code that caches data and make sure that it does not cache any of the sensitive data used by the application.

  4. Prevent caching of sensitive data. If there is code that caches sensitive data, replace it with code that meets the application requirements but does not cache sensitive data.
