Applies to

  • PCI DSS Requirement 6.6

What to Check For

Public-facing web applications should be protected by a web application firewall or be reviewed periodically. This checklist item refers to using a web application firewall.

Why

A web application firewall may prevent an attacker from successfully exploiting a vulnerability in a public-facing web application.

How to Check

Refer to the network documentation and interview the staff to identify if and how a web application firewall is installed. Examine the web application firewall and its administration interface to make sure its configuration matches the documentation. Review the web application firewall configuration to make sure that it is installed in front of a public-facing web application, and that it detects and prevents web-based attacks.

How to Fix

Choose whether to conduct regular reviews of the web application or to install a web application firewall. If you choose to install a web application firewall, choose a web application firewall that is compatible with your environment. Please refer to the vendor's documentation for detailed instructions on how to install and configure your web application firewall of choice.
