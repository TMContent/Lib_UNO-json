Applies to

  • PCI DSS Requirement 9.8

What to Do

Track off-site backups.

Why

Off-site backups should be tracked so that their location is known in the event they are needed. Tracking details and management authorization should be logged to help with a possible audit.

How

Log off-site backup activity. When logging off-site backup activity, record the following information:

  • Item name
  • Item inventory number
  • Date and time
  • Name of the person performing the action
  • Type of action
  • Destination
  • Signature of the person performing the action
  • Name of the person authorizing the action
  • Signature of the person authorizing the action
