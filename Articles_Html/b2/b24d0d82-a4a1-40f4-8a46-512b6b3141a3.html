Applies to

  • Web Application

What to Check For

Check to ensure that access to Internet-facing interfaces are restricted to port 80 for HTTP and port 443 for HTTPS (SSL).

How to Fix

Limit inbound traffic to port 80 for HTTP and port 443 for HTTPS (SSL).

Additional Resources

  • For outbound (Internet-facing) NICs, use IPSec or TCP filtering. For more information, see "How To: Use IPSec for Filtering Ports and Authentication".
