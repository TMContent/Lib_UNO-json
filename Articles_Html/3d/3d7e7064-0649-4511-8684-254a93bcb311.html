Applies to

  • PCI DSS Requirement 1.1.2.a
  • PCI DSS Requirement 1.1.2.b
  • PCI DSS Requirement 1.1.3.b

What to Check For

The network documentation should include an up-to-date network diagram that documents all connections to cardholder data, including any wireless networks.

Why

The network diagram is useful for maintenance and security tasks. The network diagram can help with identifying potential vulnerabilities in the network design, planning changes to the network and performing routine maintenance tasks. The network diagram is required to pass the PCI DSS compliance certification&nbsp;process.

How to Check

Make sure there is a staff member maintaining network documentation. Make sure that the network documentation includes an up-to-date version of the network diagram, showing all connections to the cardholder data. Compare the network diagram to the physical network to make sure the diagram corresponds to reality.

How to Fix

Inspect each network device and diagram its relationship relative to all other network devices. Make sure to document all connections to cardholder data, including any wireless networks. Designate a staff member responsible for maintaining network documentation.
