Applies to

  • C++

What to Check For

Verify that return values of functions are checked for validity.

Why

Some functions will either return a valid value or a value that represents an error condition. Checking the return values of such functions is important to assure that errors are handled. This is particularly important to memory allocation functions.

How to Check

Make a list of functions. Identify functions that return error codes. Identify code that uses these functions. Verify that code that uses these functions checks the return values for errors and handles errors appropriately.

How to Fix

Make a list of functions. Identify functions that return error codes. If possible, rewrite these functions to throw exceptions instead. Identify code that uses these functions. Add code that handles errors appropriately.
