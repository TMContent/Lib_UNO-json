Applies to

  • PHP

What to Do

Change the user's session ID when he logs in.

Why

Change session IDs during login to make sure that the session state corresponds to the authenticated user.

How

To change session IDs during login:

  1. Find login functionality. Find the code that is responsible for authentication.

  2. Change session IDs during login. Add code that invalidates the current session ID and creates a new one. For example:

    session_regenerate_id(TRUE);
