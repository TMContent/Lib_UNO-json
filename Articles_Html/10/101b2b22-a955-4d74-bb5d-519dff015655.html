Applies to

  • PCI DSS Requirement 7.1.2
  • PCI DSS Requirement 7.1.3
  • PCI DSS Requirement 7.2.2

What to Check For

Data access privileges should be assigned based on job functions.

Why

Assigning data access privileges based on job functions ensures that employees can work but cannot wage insider attacks.

How to Check

Review the data access authorization documents to make sure that privileges are explicitly defined. For a sample of users, verify that the authorization documents match the implementation.

How to Fix

Identify the job functions of each employee. Identify the data access required for each employee. Define the data access privileges for each employee based on job responsibilities.  Have the management review and authorize the data access privilege specifications. Once approved, implement the data access control systems. Attach the access authorization documents to the network documentation.
