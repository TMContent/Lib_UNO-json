Applies to

  • PCI DSS Requirement 6.1

What to Check For

The information security policy should require critical security patches to be installed within a month of being released.

Why

Installing critical security patches in a timely manner fixes known vulnerabilities before an attacker can exploit them.

How to Check

Review the information security policy to make sure it includes a point that requires critical security patches to be installed within a month of being released by the vendor.

How to Fix

Add a point to the information security policy that requires critical security patches to be installed within a month of being released by the vendor.
