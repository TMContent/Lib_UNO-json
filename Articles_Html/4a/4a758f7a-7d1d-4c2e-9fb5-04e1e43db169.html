Applies to

  • PCI DSS Requirement 2.1

What to Check For

All the manufacturer's default usernames and passwords should be changed.

Why

The default usernames and passwords are publicly available information. An attacker may be able to use the default authentication credentials to gain unauthorized access. This vulnerability is especially relevant to potential insider attacks.

How to Check

For each system component, including servers, workstations, network devices, applications, security systems, etc. obtain the default administrator username and password by using the Internet or the vendor's documentation. Try to log in to each device using the default administrator credentials.

How to Fix

If it is possible to log in to a system component using the default administrator credentials, then change the password and change the username of the administrator account, if possible. The exact procedure for changing the administrator username and/or password varies. For detailed instructions on changing the username and password, please refer to the vendor's documentation or contact technical support.
