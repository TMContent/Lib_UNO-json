Applies to

  • PCI DSS Requirement 2.1

What to Check For

Network devices should not be using the manufacturer's default usernames and passwords for administration.

Why

The default usernames and passwords for network devices are publicly available. An attacker may be able to use the default authentication credentials to change the configurations of the network devices to an insecure state. This vulnerability is especially relevant to potential insider attacks.

How to Check

For each network device, obtain the default administrator username and password by using the Internet or the vendor's documentation. Try to log in to each device using the default administrator credentials.

How to Fix

If it is possible to log in to a network device using the default administrator credentials, then change the password&nbsp;and change the username of the administrator account, if possible. The exact procedure for changing the administrator username and/or password varies by device. For detailed instructions on changing the username and password on your hardware, please refer to the vendor's documentation or contact technical support.
