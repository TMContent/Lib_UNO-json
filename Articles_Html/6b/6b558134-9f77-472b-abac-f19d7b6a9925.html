Applies to

  • PCI DSS Requirement 4.1
  • PCI DSS Requirement 4.1.e

What to Check For

If a web application uses TLS, the server should support the latest patched version.

Why

Old versions of SSL and TLS have known vulnerabilities in them, which makes them unacceptable. The latest version of TLS has addressed these vulnerabilities. New vulnerabilities may be found in the future, requiring a newer version of TLS to be used.

How to Check

Use one of the free online SSL/TLS testing tools to scan your site to determine if old versions of TLS or weak cypher suites are being used.

How to Fix

Configure your application's web server to use the latest version of TLS.
