What to Check For

If a web application uses TLS, the server should support the latest version.

Why

Old versions of SSL contain known vulnerabilities. The latest TLS version has addressed these vulnerabilities. New vulnerabilities may be found in the future, requiring a newer version of TLS to be used.

How to Check

Examine the server configuration to make sure that the latest version of TLS is used and only strong cypher suites are enabled.

How to Fix

Configure the server to use the latest version of TLS with strong cypher suites.
