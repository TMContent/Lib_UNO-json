Applies to

  • PCI DSS Requirement 8.5.4
  • PCI DSS Requirement 8.5.5

What to Do

Remove or disable accounts that have been inactive for 90 days. Also remove accounts of recently terminated employees.

Why

Accounts that are not actively used should be disabled to assure that attackers are not using them.

How

If necessary, download and install PowerShell. Download the PowerShell script from the link at the bottom of this page. Uncomment the last line of the script to make the script disable the inactive accounts. Schedule the script to run automatically every day.
