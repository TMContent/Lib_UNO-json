Applies to

  • PHP

What to Check For

Verify that directory browsing is disabled.

Why

Directory browsing allows an attacker to browse your application's directory structure in search of unprotected files and components.

How to Check

Navigate to your application's directories with a web browser. Identify whether your application's server correctly returns an HTTP error message. If it incorrectly returns the directory contents, disable directory browsing on the web server.

How to Fix

Configure the web server to disable directory browsing. The exact procedures for disabling directory browsing are different for different web servers. Please, refer to your web server's documentation for detailed instructions for disabling directory browsing.
