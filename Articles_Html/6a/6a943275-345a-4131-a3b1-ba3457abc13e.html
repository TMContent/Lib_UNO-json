What to Do

Configure the platform to timeout sessions after a period of time.

For non-web applications, implement absolute timeouts for any functionality that requires remote authentication.

Why

Limiting session lifetime makes it harder to hijack sessions.

How

Edit the platform configuration settings to set an absolute timeout for sessions. This is usually accomplished by setting a finite lifetime on the session cookie.

For non-web applications, find code that keeps track of user sessions and modify it to end sessions after a specified amount of time.
