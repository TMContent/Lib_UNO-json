Applies to

Scala and Play Framework 2.1.0+

What to Do

Make inactive sessions time-out.

Why

Make sessions expire after a period of inactivity to make it harder for an attacker to hijack a session.

How

To make inactive sessions time-out:

  1. Identify login functionality.&nbsp;Find code that handles user logins.

  2. Add code that sets the inactive session time-out period.&nbsp;There is no technical timeout for the Session. It expires when the user closes the web browser.&nbsp;

    If you need a functional timeout for a specific application, store a timestamp into the user Session and use it in the way your application needs (e.g. for a maximum session duration, maximum inactivity duration, etc.).
