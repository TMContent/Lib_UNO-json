Applies to

  • PCI DSS Requirement 7.1.4

What to Check For

The data control policy should require documented authorization for all access. The access authorization document should state granted privileges. The access privileges should be authorized by management.

Why

Access authorization documents help define privileges and identify who authorized them.

How to Check

Check the network documentation to make sure it includes access authorization documents authorized by management. Check the authorization documents to make sure they explicitly define access privileges.

How to Fix

Identify the job functions of each employee. Identify the data access required for each employee. Define the data access privileges for each employee based on job responsibilities. Have the management review and authorize the data access privilege specifications. Once approved, implement the data access control systems. Attach the access authorization documents to the network documentation. These documents may be paper or electronic.
