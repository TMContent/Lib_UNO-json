Applies To

  • ASP.NET 4.0

What to Check For

Verify that authentication is performed on the server.

Why

Perform authentication on the server to make it harder for an attacker to bypass authentication controls. It is trivial to bypass client-side web authentication because the user can edit client-side scripts.

How To Check

To make sure that authentication is performed on the server, make sure that forms authentication is being used. You can determine whether forms authentication is being used by searching Web.config files for the following string:

<authentication mode="Forms">

It is a good idea to review application code to make sure it takes full advantage of forms authentication.

How To Fix

To use forms authentication, enable and configure the following section in Web.config:

<system.web>
  <authentication mode="Forms">
     ...
  </authentication>
</system.web>
