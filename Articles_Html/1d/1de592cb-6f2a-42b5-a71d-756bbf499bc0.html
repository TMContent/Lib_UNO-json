Applies to

  • PCI DSS Requirement 2.3
  • PCI DSS Requirement 2.3.a
  • PCI DSS Requirement 2.3.c

What to Check For

All non-console administrative access should be encrypted.

Why

Encrypting remote administrative access ensures that a potential attacker cannot get administrative authentication credentials or other information about the network by eavesdropping.

How to Check

Observe an administrator log in to each system available for remote administration to make sure that a strong encryption method is invoked before the administrator is asked for a password. This should be observed over the external and local networks.

How to Fix

Use a strong encryption tunnel to secure administrative access. The exact type of tunnel depends on the application. Some common tunnels are VPN, SSH, and SSL. Most modern systems include an option or functionality to encrypt remote administrative access. For detailed instructions on encrypting administrative access on your systems, please refer to the vendor's documentation or technical support.
