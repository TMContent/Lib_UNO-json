What to Check For

Ensure that your application restricts authentication cookies&nbsp;to only&nbsp;HTTPS connections by using the requireSSL attribute.

Why

By using TLS, you prevent an attacker from reading a captured authentication cookie&nbsp;and gaining spoofed access to your application.

How to Fix

Set the secure property of the authentication cookie to ensure that browsers only send authentication cookies over HTTPS connections.

Set the secure property by using requireSSL="true" on the <forms> element as shown here:

<forms loginUrl="Secure\Login.aspx"
        requireSSL="true" ... />
