Applies to

  • PCI DSS Requirement 1.1.6.a

What to Check For

The network documentation should include descriptions of all used services, protocols and ports, and why they are necessary for business.

Why

Documenting the services that are being used is important for designing and evaluating the firewall policy. The fewer services being used, the smaller the potential target surface that is presented to the attacker. Documenting the services necessary for business helps make sure that there are no services running that are not needed.

How to Check

Check the network documentation to make sure that it includes a list and descriptions of all used services, protocols and ports that are necessary for business. Check to see that list and descriptions match reality by checking the Internet firewall configuration.

How to Fix

Identify each service that is being used. Write a description of why each service is necessary for business. Identify the port and protocol used by each service. Document the port and protocol together with the description. Make sure that the network documentation matches the configuration of the Internet firewall.
