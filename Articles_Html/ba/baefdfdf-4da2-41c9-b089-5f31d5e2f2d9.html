Applies to

  • PCI DSS Requirement 10.2.4

What to Check For

Failed access attempts should be logged.

Why

Failed access attempts may be an indication of an attack.

How to Check

This functionality should be implemented by the centralized log server solution. Examine the logs to verify that failed access attempts are logged.

How to Fix

This functionality should be implemented by the centralized log server solution. Follow the vendor's instructions to configure the log server to log failed access attempts.
