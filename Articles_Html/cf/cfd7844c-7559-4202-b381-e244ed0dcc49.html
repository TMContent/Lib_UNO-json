Applies to

  • PCI DSS Requirement 1.1

What to Check For

Check that the router configuration documentation is up-to-date. Ensure that the router configuration documentation is readily available. Verify that the router configuration documentation covers all the relevant details.

Why

Documenting the router configuration is important for maintenance and security purposes. Good router documentation makes the PCI DSS compliance certification process simpler and may be required to pass. Documentation is also very useful when making changes to the configuration and when hiring new staff.

How to Check

Make sure that the documentation for the router configuration is up-to-date, readily available, and covers all the relevant details.

How to Fix

Inspect each router and document each configuration setting thoroughly. Delegate responsibility for maintaining the network documentation up-to-date to a staff member.
