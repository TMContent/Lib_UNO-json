Applies to

  • Microsoft&#174; Windows Communication Foundation (WCF) 3.5
  • Microsoft Visual Studio&#174; 2008

What to Do

If your clients have Certificates, consider using Client Certificate Authentication.

Why

Client certificates can authenticate a client service account or multiple users to a WCF service. If you use a client certificate for each user, you can map each certificate to a Windows account.

When

Consider using this guideline when you are using Certificate Authentication.

How

Enable the Windows account mapping feature by setting the mapClientCertificateToWindowsAccount attribute to true.

<serviceCredentials>
    <clientCertificate>
        <authentication mapClientCertificateToWindowsAccount="true" />
    </clientCertificate>
</serviceCredentials>
