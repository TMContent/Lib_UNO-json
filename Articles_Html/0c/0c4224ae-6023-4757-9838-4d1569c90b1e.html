Applies to

  • Microsoft® Windows Communication Foundation (WCF) 3.5
  • Microsoft Visual Studio® 2008

Description

Implement the BeforeSendRequest method of the interface IClientMessageInspector of the message inspector to validate outbound messages on the client. This allows you to validate the message after serialization, but before sending the request to the service. 

object IClientMessageInspector.BeforeSendRequest(ref System.ServiceModel.Channels.Message request, System.ServiceModel.IClientChannel channel)
{
    validateMessage(ref request);
    return null;
}
