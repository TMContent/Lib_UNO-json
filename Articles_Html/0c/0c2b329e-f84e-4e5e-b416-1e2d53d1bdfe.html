Applies to

  • PCI DSS Requirement 11.5

What to Check For

File-integrity monitoring software should be used within the cardholder data environments.

How to Check

File-integrity monitoring may be included in the centralized log server solution, in the intrusion detection system, or in the intrusion prevention system. Identify which of these systems are deployed. Refer to the vendor's documentation or technical support of each deployed system&nbsp;for information about the file-integrity monitoring capabilities of their product. If these solutions do not provide file-monitoring capability, verify the existence of a file-integrity monitoring system by checking the logs at the centralized log server and asking the responsible staff to demonstrate the administration interface of the file-integrity monitoring solution. Verify that the locations of each of the following files are being monitored: system executables, application executables, configuration&nbsp;files,&nbsp;log files (the centralized log server should monitor its log files).

How to Fix

For any system that does not include file-integrity monitoring functionality, deploy a product that does. Configure the file-integrity monitoring software to watch the locations of the following files: system executables, application executables, configuration&nbsp;files,&nbsp;log files (the centralized log server should monitor its log files).
