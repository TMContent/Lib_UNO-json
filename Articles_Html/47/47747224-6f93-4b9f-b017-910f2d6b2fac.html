Applies to

  • PCI DSS Requirement 6.1

What to Check For

The staff should use outside resources to stay informed about the latest discovered security vulnerabilities.

Why

The information security community is very active on-line and provides many valuable resources for tracking the latest discovered vulnerabilities. Potential attackers monitor these resources, so it is important to mitigate these new threats before they are exploited.

How to Check

Interview the staff and ask them what outside resources they use to stay informed about the latest discovered security vulnerabilities.

How to Fix

Add a requirement to the information security policy for the responsible staff to use outside resources to stay informed about the latest discovered security vulnerabilities. Instruct the staff to review on-line security vulnerability tracking resources periodically. Instruct at least one staff member to sign up for the BugTraq mailing list and other similar mailing lists as appropriate. Update configuration standards in the network documentation as appropriate to mitigate the newly discovered vulnerabilities.

Additional Resources

  • To sign up for the BugTraq mailing list, please visit "Security Focus" at http://www.securityfocus.com
  • For a list of recently discovered vulnerabilities, please see "Security Focus Vulnerabilities" at http://www.securityfocus.com/vulnerabilities
  • To check the latest public exploits, please see "PacketStorm" at http://www.packetstormsecurity.org/exploits100.html
