Applies to

  • PHP

What to Do

Configure PHP to make sessions time-out after a certain period of time, even if the user is active.

Why

Limiting session life-time makes it harder to hijack sessions.

How

To set an absolute session time limit, configure the php.ini file to include the following directive:

session.cookie_lifetime = 60000

The snippet above sets the maximum session life-time to 60000 seconds by limiting the life-time of the session cookies.
