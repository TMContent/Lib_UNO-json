Applies to

  • C++

What to Check For

Verify that variables are initialized before being used.

Why

Using variables before they are initialized may result in unexpected behavior.

How to Check

Identify code that declares variables. Verify that all variables are initialized before being used. Most compilers warn about using unitialized variables.

How to Fix

Identify code that declares variables. Add code that initializes variables before they are used.
