What to Check For

Verify that TLS is used for all sensitive pages.

Why

TLS should be used to protect all pages that handle sensitive data, including all pages that use authentication or authorization features, to prevent unauthorized information disclosure.

How to Check

To verify that TLS is required for all sensitive pages:

  1. Identify sensitive pages. Make a list of all pages that handle sensitive information, including all pages that use authentication or authorization features.

  2. Verify that TLS is required for all sensitive pages. Browse to each sensitive page to verify that TLS is used. Try accessing each page via HTTP and via HTTPS to make sure that sensitive pages are only available via HTTPS.

How to Fix

The simplest approach for enabling TLS for all sensitive pages is to use TLS for the entire site. Enabling TLS for the entire site may require additional computing resources, so for performance reasons, it may be acceptable to enable TLS only for sensitive pages. To enable TLS only for sensitive pages:

  1. Identify sensitive pages. Make a list of all pages that handle sensitive information, including all pages that use authentication or authorization features.

  2. Enable TLS for each directory containing sensitive pages. The exact instructions for doing so are different for different web servers. Use the web server administration interface or edit the server configuration files to enable TLS for all folders containing sensitive pages.

  3. Redirect non-TLS requests to TLS-protected pages. This can be accomplished by editing the .htaccess file in each directory with sensitive pages and adding the folowing lines:

    RewriteEngine On
    RewriteCond %{HTTPS} !on
    RewriteRule (.*) https://%{HTTP_HOST}%{REQUEST_URI}

    4. The $_SERVER['HTTPS'] variable can be used to programmatically determine if TLS is being used and take action accordingly.

Transport Layer Protection Guidelines

  • Use TLS
  • Use TLS for All Sensitive Pages
  • Use Only Strong TLS Algorithms
  • Use Valid TLS Certificates

Transport Layer Protection Checklists

  • TLS Is Enabled
  • TLS Is Used for All Sensitive Pages
  • Only Strong TLS Algorithms Are Used
  • Valid TLS Certificates Are Used
