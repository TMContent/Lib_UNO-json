Applies to

  • PCI DSS Requirement 5.1.1

What to Do

Use an anti-virus solution that can handle all known malware types.

Why

An anti-virus solution has to handle all known types of malware to be practical for improving the overall security posture.

How

If your anti-virus does not adequately handle all known types of malware threats, install and configure an anti-virus that does. In some cases, updating the anti-virus to the latest version may be sufficient. The following are known malware types:

  • Trojans
  • Viruses
  • Worms
  • Spyware
  • Adware
  • Rootkits
  • Exploits
  • Hacking Tools
