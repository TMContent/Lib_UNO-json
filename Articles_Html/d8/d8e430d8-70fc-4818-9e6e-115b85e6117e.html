Applies to

  • PCI DSS Requirement 6.3

What to Check For

Security procedures should be included throughout the development life cycle.

How to Check

Review the development life cycle documentation to make sure it includes security procedures for each distinct stage.

How to Fix

Identify the steps in the software development life cycle. TeamMentor materials can be used to integrate security into the SDL. Design, implement and document the relevant security procedures for each distinct stage. The exact stages and procedures depend on the choice of the software development model and the application being developed.
