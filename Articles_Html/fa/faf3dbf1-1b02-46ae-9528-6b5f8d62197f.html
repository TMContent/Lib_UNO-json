Applies To

  • PCI DSS Requirement 6.4.4

Policy Statement

Remove test data and accounts before deployment.

Policy Purpose

Test accounts and data have to be removed before deployment because they might give away information about the application that can be used during attacks. Test accounts might also be used to compromise the application if they have weak authentication credentials.
