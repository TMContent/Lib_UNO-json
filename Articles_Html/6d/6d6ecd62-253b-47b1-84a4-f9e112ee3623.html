Applies to

Scala and Play Framework 2.1.0+

What to Check For

Verify that authentication controls are implemented on the server, rather than in client-side code.

Why

Use server-side authentication because client-side authentication is easy to bypass.

How to Check

To verify that server-side authentication is used:

  1. Identify authentication controls. Find code that is responsible for authentication.

  2. Verify that authentication controls are implemented on the server. Review authentication code to make sure that authentication is performed on the server.

How to Fix

To implement server-side authentication, use authentication controls provided by the framework, or use any well-known third-party authentication library. If you have to implement your own authentication controls, design them to perform authentication on the server-side.
