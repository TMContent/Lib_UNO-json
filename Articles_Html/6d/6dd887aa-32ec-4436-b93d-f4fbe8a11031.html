Applies To

  • ASP.NET 4.0

What to Check For

Verify that sessions expire when users log out.

Why

Making the session expire when the user logs out makes it harder for an attacker to hijack the session.

How To Check

To verify that sessions expire when users log out:

  1. Identify logout functionality. Find code that is responsible for the logout functionality.

  2. Verify that sessions expire upon logout. Look for the following code:

    Session.Abandon();

How To Fix

To make sessions expire when users log out:

  1. Identify logout functionality. Find code that is responsible for the logout functionality.

  2. Add code to make sessions expire. Add the following code to the logout function to make the session expire:

    Session.Abandon();
