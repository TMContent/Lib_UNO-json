Applies to

  • PCI DSS Requirement 4.2
  • PCI DSS Requirement 4.2.a
  • PCI DSS Requirement 4.2.b

What to Check For

The information security policy should include a point that forbids using messaging software to send unencrypted PANs.

Why

Not using messaging programs to send unencrypted PANs prevents the possibility of them being read by an attacker if intercepted.

How to Check

Review the information security policy to make sure it includes a point that prohibits sending unencrypted PANs using messaging software. Interview staff members and ask them if they use messaging software to send unencrypted PANs. Interview staff members and ask them if they use messaging software to send encrypted PANs. If messaging software is used to send encrypted PANs, verify that the PANs are secured using strong cryptography during transmission.

How to Fix

Add a point to the information security policy that forbids employees from sending unencrypted PANs using instant messaging software. Instruct staff members to not use instant messaging software to send unencrypted PANs.
