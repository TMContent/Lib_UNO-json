Applies to

  • PCI DSS Requirement 3.6.2

What to Check For

Key management procedures should require secure key distribution.

How to Check

Check the information security policy to make sure the key management procedures require secure key distribution.

How to Fix

Design and document the key management procedures to require secure key distribution. Include the key management procedures in the information security policy. To qualify, the key management procedures should explain how the keys are distributed and what limitations make them secure. One example of a secure key distribution procedure is only distributing the keys on the application server.
