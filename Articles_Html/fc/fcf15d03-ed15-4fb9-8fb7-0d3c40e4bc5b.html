Applies to

  • PCI DSS Requirement 7.2.1
  • PCI DSS Requirement 7.2.3

What to Do

Cover all system components using access controls.

Why

Using access control systems to secure all system components ensures that there is no information that can be retrieved by unauthorized personnel.

How

Identify all computers and other system components. Examine each system component to make sure it implements an access control system. Most modern Operating Systems and network devices implement access control. If a system component does not implement access control, it cannot be used within PCI DSS scope. Set a "deny-all" default on all access control systems. Examine the access authorization documents in the network documentation. Implement the access privileges defined in the access authorization documents. The access authorization documents only allow data access that is required for each employee's job function, so they comply with the "least-privilege" principle.
