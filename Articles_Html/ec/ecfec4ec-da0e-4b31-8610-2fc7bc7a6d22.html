Applies to

  • PCI DSS Requirement 1.2.1.a
  • PCI DSS Requirement 1.2.1.b

What to Check For

Unless a network transmission is required for business operations, it should be denied by default. An explicit "deny all" or implicit "deny after allow" setting should be used on the Internet firewall.

Why

Allowing unauthorized connections may allow an attacker to conduct an attack and extract sensitive information.

How to Check

Check the firewall configuration to make sure that it uses either an explicit "deny all" setting or an implicit "deny after allow" setting.&nbsp;Perform a port scan of the company network from the Internet to make sure that only authorized ports are open.

How to Fix

Configure the firewall using either the "deny all" setting or the "deny after allow‚Äù setting. When using the "deny all" rule, set it after the rules that allow services necessary for business operations. If your firewall supports implicit "deny after allow" functionality, you may use it instead.
