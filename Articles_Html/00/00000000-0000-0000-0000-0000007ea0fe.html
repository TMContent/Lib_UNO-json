Applies To

  • Android applications that offer file sharing.

What to Do

Disable file sharing by default in applications that offer file sharing.

Why

Disable file sharing to prevent unauthorized access. If file sharing is necessary, it should only be enabled when it is being used.

How

To disable file sharing by default.

  1. Identify file sharing functionality. Determine whether the application implements file sharing. If so, find the code that implements it.
  2. Disable file sharing by default. Edit the code that implements file sharing to disable file sharing by default. Provide controls to the user to enable file sharing when needed. Encrypt network connections used for file sharing. Protect file sharing with usernames and passwords and to allow the user to set the usernames and passwords that are allowed to access shared files.

Authorization Guidelines

  • Disable File Sharing by Default
  • Use Role-based Access Control

Authorization Checklists

  • File Sharing Is Disabled by Default
  • Role-based Access Control Is Used
