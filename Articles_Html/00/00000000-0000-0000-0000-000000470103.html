What to Do

Sign application updates.

Why

Cryptographically signing applications assures that the application binary has not been tampered with and that it came from a trusted source. In practice, updates to iOS applications are distributed as full new versions rather than patches. iOS application updates have to be signed to run, just like entirely new applications have to be signed.

How

The procedure for signing iOS application updates is the same as the procedure for signing iOS applications in general.
