Applies To

  • iOS applications that handle XML documents.

What to Do

Disable parsing of external XML entities.

Why

Parsing external entities in XML documents may result in XML injection vulnerabilities.

How

There are two common libraries to parse XML in iOS: NSXMLParser and LibXML2.

  1. NSXMLParser. Parsing of external entities is disabled by default in NSXMLParser. Don't explicitly enable parsing of external entities - avoid using the setShouldResolveExternalEntities option.
  2. LibXML2. Parsing of external entitites needs to be manually disabled in LibXML2. To disable parsing of external entities, use the xmlSetExternalEntityLoader method of the parser class to set the external entity loader to be a function that doesn't do anything.
