What to Check For

Verify that file sharing is disabled in applications by default.

Why

Disable file sharing to prevent unauthorized access. If file sharing is necessary, it should only be enabled when it is being used.

How to Check

To verify that file sharing is disabled:

  1. Identify file sharing functionality. Determine whether the application implements file sharing and if so, find the code that implements it.
  2. Verify that file sharing is disabled by default. Check the default settings to make sure that file sharing is disabled by default.

How to Fix

To disable file sharing by default.

  1. Identify file sharing functionality. Determine whether the application implements file sharing and if so, find the code that implements it.
  2. Disable file sharing by default. Edit the code that implements file sharing to disable file sharing by default. Provide controls to the user to enable file sharing when needed. It is recommended to encrypt network connections used for file sharing. It is also recommended to protect file sharing with usernames and passwords and to allow the user to set the usernames and passwords that are allowed to access shared files.

Authorization Guidelines

  • Disable File Sharing by Default
  • Use Role-based Access Control

Authorization Checklists

  • File Sharing Is Disabled by Default
  • Role-based Access Control Is Used
