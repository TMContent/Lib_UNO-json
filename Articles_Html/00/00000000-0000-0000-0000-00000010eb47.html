Applies To

  • HTML5

What to Do

Use the textContent method instead of innerHTML to assign values to DOM elements.

Why

Using the textContent method instead of innerHTML prevents cross-site scripting (XSS) attacks, because values assigned using textContent will not be executed as script.

How

To use the textContent property instead of innerHTML:

  1. Find all references to innerHTML. Search the application code for all references to the innerHTML property. Create a spreadsheet to list all code that uses the innerHTML property to assign values to DOM elements.
  2. Use textContent instead of innerHTML. Using the spreadsheet as a guide, for all code that uses innerHTML to assign values, change it to use textContent instead. Simply replace references to innerHTML with textContent. For most applications, no further changes should be required.

Web Messaging Guidelines

  • Do Not Evaluate Messages as Code
  • Use element.textContent to Assign Values to DOM Elements
  • Explicitly State the Expected Origin with postMessage
  • Validate Data Received by Event Handlers
  • Validate the Origin Attribute Against the FQDN
  • Check The Origin Attribute of Messages
