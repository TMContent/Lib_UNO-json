What to Check For

Verify that location data is not logged.

Why

Logging location data is likely to raise privacy concerns among users.

How to Check

To verify that location data is not logged:

  1. Identify whether the application uses location services.
  2. Determine how location services are used. Trace location data from source to sink and record each operation performed on it.
  3. Verify that location data is not logged. Examine the trace of location data from source to sink to make sure that location data is not being logged either on the device or on any remote server.

How to Fix

If your application uses location data, do not store it in files and do not send it to remote servers.

Privacy Guidelines

  • Do Not Cache Sensitive Data
  • Do Not Log Location Data
  • Do Not Log Sensitive Data
  • Delete Cached Data after Use

Privacy Checklists

  • Sensitive Data Is Not Cached
  • Location Data Is Not Logged
  • Sensitive Data Is Not Logged
  • Cached Data Is Deleted after Use
