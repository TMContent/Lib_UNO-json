Applies To

  • All iOS applications.

What to Do

Use stack smashing protection offered by the compiler.

Why

Stack cookies help mitigate buffer overflow vulnerabilities, which can allow attackers to execute commands remotely.

How

To use stack cookies:

  1. Open XCode. Launch the XCode IDE.
  2. Add compiler flags. Add the -fstack-protector-all flag to each file that is to be compiled in the XCode "Compile Sources" window in the "Compiler Flags" column.

Memory Management Guidelines

  • Compile Applications with Stack Cookies Enabled
  • Do Not Reference Freed Objects
  • Use Automatic Reference Counting

Memory Management Checklists

  • Applications Are Compiled with Stack Cookies Enabled
  • Freed Objects Are Not Referenced
  • Automatic Reference Counting Is Used
