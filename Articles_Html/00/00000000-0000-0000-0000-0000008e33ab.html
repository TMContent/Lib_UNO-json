What to Check For

Verify that code is obfuscated with ProGuard.

Why

Obfuscate the application binary with ProGuard to make it more difficult to reverse engineer.

How to Check

To verify that code is obfuscated with ProGuard:

  1. Verify that ProGuard is configured. Make sure the proguard.cfg file is present. Examine the proguard.cfg file to make sure that it suits the application.
  2. Verify that ProGuard is enabled. Verify that the proguard.config property in the project.properties file points to the location of the proguard.cfg file.
  3. Verify that the application is built in release mode. Examine the build environment to make sure that it is set to release mode.

How to Fix

To obfuscate code with ProGuard:

  1. Configure ProGuard. Edit the proguard.cfg file to suit your application.
  2. Enable ProGuard. Change the proguard.config property in the project.properties file to point to the location of the proguard.cfg file.
  3. Enable release mode. ProGuard is executed automatically by the Android build system when it is used to build applications in release mode.
  4. Build the application. Build your application in release mode to obfuscate the binary with ProGuard.
