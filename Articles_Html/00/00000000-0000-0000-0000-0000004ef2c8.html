What to Do

Sign updates to the application.

Why

Sign updates to the application to help verify that the updates came from a trusted party.

How

Signing updates is an important principle in software development; it prevents malicious software from masquerading as updates to legitimate software. In practice, updates to mobile software are usually distributed as complete new versions, rather than as patches to the already installed software. The updated software package is usually signed, because otherwise it won't be allowed on the most popular distribution channels. Therefore, the process for signing mobile software and updates is usually already established by software developers. It is still important to be aware of the importance of signing software and software updates.
