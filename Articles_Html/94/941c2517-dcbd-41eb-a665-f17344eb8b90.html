What to Check For

Access to logs should be logged.

Why

Logging access to the logs helps investigate potential tampering with the logs.

How to Check

This functionality should be implemented by the centralized log server solution. Examine the logs to verify that access to logs is logged.

How to Fix

This functionality should be implemented by the centralized log server solution. Configure the centralized log server to log access to logs. Please refer to the vendor documentation or contact tech support for detailed instructions on configuring your centralized log server to log access to the log files.
