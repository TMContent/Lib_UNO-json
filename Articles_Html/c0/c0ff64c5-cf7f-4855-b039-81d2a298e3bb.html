What to Do

Deny all access by default.

Why

All access should be denied by default to make sure that only authorized users gain access to sensitive information.

How

To deny all access by default:

  1. Identify available authorization functionality. Your application may have a custom authorization library or it may use access controls provided by the framework.

  2. Configure access controls to deny all access by default. If your application uses a custom authorization library, please refer to its documentation for instructions on how to configure it to deny all access by default. If your application uses access controls provided by the platform, please refer to the platform documentation for instructions on how to configure it to deny all access by default.
