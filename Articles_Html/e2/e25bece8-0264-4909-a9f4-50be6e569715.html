Applies to

  • C++

What to Check For

Verify that string length is validated when strings are passed between functions.

Why

Sending string parameters to functions is one of the most common sources of buffer overflow vulnerabilities. Validating the sizes of strings before using them prevents buffer overflows.

How to Check

Identify functions that accept strings as arguments. Verify that input and data validation functions are called before these strings are used. Verify that these data and input validation functions check string sizes.

How to Fix

It is recommended to use a centralized input and data validation class or library. Add code to that class or library to validate string sizes.

Identify functions that accept strings as arguments. Add code to these functions that calls validation routines to verify that string size is acceptable.
