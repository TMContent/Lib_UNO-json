Applies to

  • PCI DSS Requirement 5.1

What to Check For

An anti-virus solution should be deployed on all systems that are commonly targeted by malware.

Why

An anti-virus solution helps stop large-scale automated attacks.

How to Check

Identify all computers that use operating systems that are commonly affected by malware. Take a sample of these computers and verify that they have anti-virus software installed, if anti-virus software for these systems exists.

How to Fix

Identify all computers that use operating systems that are commonly affected by malware. Choose an anti-virus package that can be installed on these systems. When choosing an anti-virus package, consider the following capabilities:

  • Automatic updates
  • Automatic scans
  • Log retention for at least a year
  • Ability to handle all known malware threat types
  • Integration with the centralized log server

Install and configure the chosen anti-virus package on all the compatible systems.
