Applies to

  • Microsoft® Windows Communication Foundation (WCF) 3.5
  • Microsoft Visual Studio® 2008

What to Do

If you need to validate inbound messages on the client, implement the AfterReceiveReply method of the interface IClientMessageInspector of the message inspector.

When

Consider using this guideline when you want to validate the client message after client response has arrived and before deserialization and returning the data to the client application.

How

If you need to perform validation on client of inbound messages, implement the AfterReceiveReply of the interface IClientMessageInspector of the message inspector. This will validate the message after client response has arrived and before deserialization and returning the data to the client application.

Following example shows how to implement the AfterReceiveReply method.

void IClientMessageInspector.AfterReceiveReply(ref System.ServiceModel.Channels.Message reply, object correlationState)
 {
     validateMessage(ref reply); 
 }
