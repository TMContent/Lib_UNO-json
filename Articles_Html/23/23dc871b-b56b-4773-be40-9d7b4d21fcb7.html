Applies to

  • PCI DSS Requirement 2.2.3.b

What to Do

Include common security settings in system configuration standards.

Why

Including common security settings in the system configuration standards helps make sure that the security settings are consistent across the network.

How

Identify each type of system on the network. For each type of system, identify security features and configuration options it provides. Decide on the appropriate settings for the security features and capabilities of each system type. Document the chosen settings of the security features of each type of system in the system configuration standards.
