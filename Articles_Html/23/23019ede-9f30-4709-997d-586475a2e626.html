What to Check For

Verify that strong symmetric encryption is used.

Why

Using strong encryption algorithms helps prevent unauthorized disclosure of sensitive information.

How to Check

To verify that strong symmetric encryption is used:

  1. Identify sensitive data. Find out what data should be encrypted using symmetric encryption.

  2. Identify code that handles sensitive data. Find code that handles the sensitive data.

  3. Verify that strong encryption algorithms are used. Examine code that handles sensitive data to make sure that it uses AES-256 whenever symmetric encryption is needed.

How to Fix

To use strong symmetric encryption:

  1. Use Java cryptographic APIs. Java cryptographic APIs are available in the following classes: java.security.Security, javax.crypto.Cipher, and javax.crypto.spec.SecretKeySpec.

  2. Write cryptographic utility functions that use AES-256. Write functions that generate keys, encrypt, and decrypt data. For an example, please see the link called "Encryption and Decryption with AES" at the bottom of this page.

  3. Use the utility functions. Use the cryptographic utility functions whenever data needs to be encrypted or decrypted using symmetric cryptography.
