Applies to

Scala and Play Framework 2.1.0+

What to Do

Use valid TLS certificates.

Why

Using valid TLS certificates helps prevent phishing attacks by assuring visitors that your site is secure and trusted.

When

Always use valid TLS certificates.

How

To use a valid TLS certificate:

  1. Purchase a valid TLS certificate. You may purchase a valid TLS key from VeriSign, Thawte, or your web hosting provider. Follow the vendor's instructions for purchasing the TLS certificate. The certificate is usually a .cer file. Make sure the certificate matches all domains used by the application.

  2. Import the valid TLS certificate. The exact instructions for importing TLS certificates are different for different application servers. Please refer to your application server's documentation or technical support for detailed instructions on how to install TLS certificates.

  3. Configure the web application server to use TLS. The exact instructions for configuring web application servers to use TLS are different for different platforms. Please refer to your application server's documentation or technical support for detailed instructions on how to install TLS certificates.
