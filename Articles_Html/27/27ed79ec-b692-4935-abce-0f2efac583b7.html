Applies to

  • PCI DSS Requirement 2.1.1.a

What to Check For

Check that the wireless access keys are changed whenever an employee that has had access to the keys leaves the company. Verify that the information security policy states the requirement to change the keys when an employee that has had access to them leaves the company.

Why

Wireless keys should be changed when an employee leaves to prevent insider attacks or the former employee disclosing the key to a potential attacker.

How to Check

Examine the information security policy to make sure it includes a requirement to change the wireless keys whenever an employee who has had access to the keys leaves the company. Interview the staff member(s) responsible for the security of the wireless network to make sure that the keys are changed when an employee leaves the company.

How to Fix

Add a statement to the information security policy that requires the wireless access keys to be changed whenever an employee that has had access to the keys leaves the company. If an employee that has had access to the wireless keys has left, change the wireless access keys.
