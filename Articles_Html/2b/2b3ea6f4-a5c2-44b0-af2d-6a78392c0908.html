Applies to

  • PCI DSS Requirement 1.1.1

What to Check For

The network documentation and the Information Security policy should include a requirement for a formal process for testing changes to the router configuration, as well as directions for the procedure to test those changes.

Why

Changes to the router configuration must be tested to make sure that vulnerabilities are not introduced inadvertently by the changes.

How to Check

Examine the network documentation and the Information Security policy to make sure they include a requirement and a description of a process to test changes to the router configuration.

How to Fix

Add a point to the Information Security policy to require following a specific process to test the router configuration whenever changes are made. Describe a process for testing changes to the router configuration in the network documentation. Perform a test run of the testing process to make sure that it is adequate for your network.
