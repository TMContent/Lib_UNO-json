Applies to

  • Microsoft® Windows Communication Foundation (WCF) 3.5
  • Microsoft Visual Studio® 2008

Description

Use the following steps to enable a WMI provider for your service:

  1. Open the web.config file of the WCF service using the Configuration editor tool (SvcConfigEditor.exe).
  2. In the Configuration editor, navigate to the Diagnostics node, and click the Enable WMI Provider link. 
      <diagnostics wmiProviderEnabled="true">
  3. To view the WMI trace information you need to install WMI CIM Studio to view the WMI interactions. It is an ActiveX component that plugs into Internet Explorer. You can get this as a free download available from Microsoft.
