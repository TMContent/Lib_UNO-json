What to Check For

Verify that session management offered by the framework is used.

Why

The framework works on a lower level than the application and is therefore better suited to provide session management. Most mature web application frameworks offer secure session management functionality.

How to Check

To verify that session management offered by the framework is used:

  1. Identify session management functionality. Find code that handles session management.

  2. Verify that session management offered by the framework is used. Review code that uses session management to make sure it uses functionality offered by the framework.

How to Fix

To use session management offered by the framework, refer to the documentation for your chosen framework.
