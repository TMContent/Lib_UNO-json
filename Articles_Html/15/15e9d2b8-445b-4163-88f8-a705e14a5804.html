Applies to

Scala and Play Framework 2.1.0+

What to Do

Mask all password fields. Use input controls that hide user input for password fields by covering user input with a "mask", typically a character, such as "*". When a user enters a password "password", the masked field may look like "********".

Why

Masking password fields helps protect user passwords from someone that may be looking at the screen.

How

To mask all password fields:

  1. Identify all password fields in the application.&nbsp;Review all the pages in the application to find all password fields.

  2. Use password fields that mask user input.&nbsp;There are different approaches to masking user input:

    • Masking password fields is usually done by using a "password" type HTML input field, as follows:

      <form>
      <input type="password" name="pass" autocomplete="off" />
      </form>

    • Set the "autocomplete" attribute to "off" to prevent the user's browser from caching the entered password.

    • If you are using third party controls for password fields, choose ones that mask user input.

    • If you are implementing custom controls for password fields, implement these controls in such a way that they mask user input.
