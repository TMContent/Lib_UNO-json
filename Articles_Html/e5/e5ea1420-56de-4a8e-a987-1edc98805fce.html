Applies to

  • PCI DSS Requirement 3.1.b

What to Check For

The information security policy should include provisions for data disposal.

How to Check

Check the information security policy to make sure that it includes provisions for data disposal.

How to Fix

Identify the method for disposing of data that is older than required by the data retention policy. Describe in detail the chosen method for data disposal and include the description in the information security policy.
