Applies to

  • PCI DSS Requirement 5.2
  • PCI DSS Requirement 5.2.b
  • PCI DSS Requirement 5.2.c

What to Check For

The anti-virus software should be kept up-to-date through regular, automatic updates.

How to Check

Find the master installation of the anti-virus software and make sure that it has the capability to automatically update itself. Take a sample of systems that have the anti-virus software installed and check the options to verify that automatic updates are enabled. To check the anti-virus options, locate the anti-virus administration interface. Some anti-virus packages have a control panel on the systems where they are deployed and some anti-virus suites offer a centralized control panel. The number of options offered by the anti-virus interface is usually small enough that a person with an IT background can figure out what all of them are and how to use them. The administration interface usually has an option to enable automatic updates and configure their frequency. Check the logs to make sure that the automatic updates are taking place.

How to Fix

Verify that it is possible to run automatic updates with your anti-virus by referring to the vendor documentation or technical support. The documentation or technical support should provide the exact instructions on how to implement automatic updates. Automatic updates are a basic feature of modern anti-virus packages. If your anti-virus does not offer this feature, please consider installing and configuring a different anti-virus.
