What to Check For

Verify that error handling functions are centralized, for example in a separate class, or a set of classes.

Why

Centralized error handling is easier to test and implement correctly. Handling errors is important for security, so better error handling improves security.

How to Check

Look for code that handles errors. Verify that functions and classes are written specifically for handling errors, instead of handling errors in code that may cause them.

How to Fix

Write utility classes and functions for handling errors. Use these functions instead of handling errors in code that may cause them.
