Applies to

  • PCI DSS Requirement 1.3
  • PCI DSS Requirement 1.3.1

What to Check For

The network should have a segment designated as the DMZ. The DMZ is the network segment that has Internet access and may handle incoming Internet traffic. There may be more than one DMZ.

Why

The DMZ is a defense-in-depth measure used to separate systems that are open to attacks from those that hold sensitive information. When a DMZ is implemented, even if the systems in the DMZ are compromised, the attacker does not have full access to the sensitive information.

How to Check

Check the network diagram to ensure that a network segment is designated as the DMZ. Check that the physical network matches the network diagram.

How to Fix

Designate a network segment as the DMZ. Install and configure a firewall between the DMZ and the Internet. Install and configure a firewall between the DMZ and the internal network segment. Document the network.
