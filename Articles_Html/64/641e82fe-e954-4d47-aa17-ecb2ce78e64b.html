Applies to

  • PCI DSS Requirement 1.1.3

What to Do

Install a firewall between the DMZ and the internal network.

Why

The DMZ has machines accessible from the Internet and is therefore subject to attack. In the event that machines in the DMZ are compromised, the firewall should protect the cardholder data environment from the DMZ. Installing a firewall between the DMZ and the cardholder data environment is a defense-in-depth measure - it offers an extra layer of security in the event that some of the network is compromised.

How

Install a firewall between the DMZ and the cardholder data environment. Install a firewall between the DMZ and any other internal network segment. Configure the firewalls in accordance with the firewall configuration standards. Add any new firewalls to the network diagram in the network documentation.
